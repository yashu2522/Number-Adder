# Number Adder ➕

A simple web application that allows users to input two numbers and calculate their sum instantly.

This project was built as part of revising JavaScript fundamentals and DOM manipulation.

## 🚀 Tech Stack

* HTML5
* CSS3
* JavaScript (DOM Manipulation)
* Font Awesome (for icons)

## ✨ Features

* Input two numbers easily
* Instant calculation of sum
* Clean and centered UI design
* Responsive layout
* Interactive button with hover effects

## 📚 What I Practiced

* Handling user input fields
* DOM manipulation using `getElementById`
* Event handling with `addEventListener`
* Type conversion using `parseFloat()`
* Updating content dynamically with `innerHTML`

## 📌 Future Improvements

* Add input validation (handle empty fields)
* Show error messages for invalid input
* Add support for more operations (subtract, multiply, divide)
* Improve UI with animations and transitions

## 📁 Project Status

Completed as a revision project to strengthen JavaScript basics and user interaction handling.
